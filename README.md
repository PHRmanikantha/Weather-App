#### Description:
This is my **CS50P Final Project** — a simple Python-based **Weather Forecast App** that fetches **real-time weather data** using the **OpenWeather API**.

The app allows the user to:
- Enter any city name 🌎
- Get **temperature**, **humidity**, **wind speed**, and **weather condition** 🌤️
- Handles invalid city names gracefully ❌

---

## ✨ Features
- ✅ Real-time weather data via API
- ✅ Simple command-line interface
- ✅ Error handling for invalid inputs
- ✅ Beginner-friendly, but looks professional

---

## 🛠️ Technologies Used
- **Python 3** 🐍
- **Requests Library** 🌐
- **OpenWeather API** ⛅
